#Configuration

1. System	: /etc/gitconfig
2. Global	: ~/.gitconfig
2. Local	: .git/config
	
**Globally :**
	git config --global user.name "username"
	git config --global user.email "user email"
	
**Locally :**
	git config user.name "username"
	git config user.email "user email"

	git config --global color.ui true
	
#Initializing

	git init
	
#Status - add

	git status
	git add README
